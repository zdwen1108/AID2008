## git 的使用
> 基本命令
``` git config
    git add 添加到暂存区
    git commit 提交
    git tag 标签
    git reset 版本的跳转
    git stash 本地工作区
    git branch 分支管理
   
```
> githup使用
```
    git remote 连接远程仓库
    git clone 下载远程代码
    git push 上传本地代码
    git pull/fetch 拉取远程代码
```
